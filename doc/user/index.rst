In order to ease the installation of these software on multiple operating systems, the **Conda** package and environment management system is used.
To install **Conda**, you can download and execute the following scripts:

* :download:`install.bat`, if you are on **Windows** operating systems.
* :download:`install.sh`, if you are on **Unix** operating systems.

    For more documentation on each software included in this suite, please refer to its own documentation.
